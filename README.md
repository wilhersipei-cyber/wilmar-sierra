# Portafolio Profesional – Wilmar Sierra

Este repositorio contiene el código fuente de mi página web profesional, donde presento mi experiencia, formación técnica y trayectoria en el sector industrial y energético.

La página está diseñada como un portafolio profesional enfocado en **operación de plantas industriales, generación de vapor y utilidades energéticas**.

## Perfil Profesional

Soy tecnólogo en Procesos de la Industria Química con más de 10 años de experiencia en operación de plantas industriales, sistemas de generación de vapor y utilidades energéticas.

A lo largo de mi trayectoria he trabajado en operación de:

- Calderas acuotubulares y pirotubulares
- Turbogeneradores
- Compresores centrífugos Siemens e Ingersoll Rand
- Sistemas de tratamiento de agua por ósmosis inversa
- Operaciones de transferencia de hidrocarburos
- Monitoreo de variables de proceso en campo y cuarto de control

Mi experiencia se ha desarrollado en entornos industriales de alta exigencia técnica, siempre bajo lineamientos de **seguridad industrial (HSE)** y continuidad operativa.

## Contenido del sitio web

El sitio incluye las siguientes secciones:

- Presentación profesional
- Experiencia laboral
- Equipos y sistemas operados
- Formación académica
- Información de contacto

## Tecnologías utilizadas

El sitio fue desarrollado utilizando tecnologías web básicas:

- HTML5
- CSS3
- JavaScript

No utiliza frameworks externos, lo que permite un sitio ligero, rápido y fácil de mantener.
